## 1. `Tao-Research/out/2606.18009v1.pdf`

Title: **Making Sense of Symbols: Yin and Yang in Zurich**  
Author: Frank Schweitzer  
Status after this pass: `COMPLETE_FIRST_PASS`

### Paper-level read

The paper reconstructs the Taijitu/Yin-Yang symbol from a quantitative model of daytime duration at Zurich latitude. The core variable is an excess daytime/nighttime fraction over the equinox-to-solstice difference. The S-curve is not treated as a raw plot of daylight length. Its stronger interpretation is stability: it marks where Yin/Yang predominance changes.

### Mechanism extraction

The most important section is the stability discussion. The paper says the symbol does not simply reflect actual day/night values. It reflects the **conditions under which subtle prevalence changes**. The seesaw analogy gives the exact pattern:

```text
continuous hidden control parameter changes
stable/metastable visible regime persists
critical threshold crossed
visible dominance changes
transition quadrant appears
new stable/metastable regime holds
```

The paper also maps Yijing line states onto quadrants:

```text
Quadrant I   → young Yin  = 8, not yet critical
Quadrant II  → old Yin    = 6, transforming into Yang
Quadrant III → young Yang = 7, just appeared
Quadrant IV  → old Yang   = 9, transforming into young Yin
```

The strongest phrase for the Phase stack is the paper's distinction between activity/change and tranquility/rest: rest does not mean nothing happens. A continuous process continues **under the hood**, but the system remains visibly stable because its time has not yet come.

### Reunification

This is not merely a Taijitu geometry lead. It is a regime-threshold model:

| Phase role | External analogue in 2606 |
|---|---|
| Q | quadrant / young-old phase position / control-parameter sector |
| B | hidden continuous change inside the current regime |
| floor/capacity | critical stability threshold |
| blocked visible transition | old Yin / old Yang transformation phase |
| L | regime change after loss of stability |

This is now one of the cleanest external models for why a system can be locally active while projection appears still. It protects the B/Q coupling correction: B can be doing refinement under the hood while visible Q-state has not yet crossed the threshold.

### Status

High-value external analogue for **Q-admitted phase capacity** and visible transition after hidden saturation.

## 2. `Prospect-Leads/philosophies-10-00111.pdf`

Title: **Research on the Mathematical Principles of Chinese Philosophy from the Body Dimension in Traditional Chinese Medicine**  
Authors: Haijin Xie, Ruifeng Yan  
Status after this pass: `COMPLETE_FIRST_PASS`

### Paper-level read

This paper is a 2025 review/argument connecting Yi Jing, traditional Chinese medicine, zhishu, yinyang, wuxing, luoshu, jiugong, and medical calculation. Its value is not that it proves Phase Calculus. Its value is that it contains a surprisingly explicit **state-conditioned selection grammar**.

### Core extractions

#### 2.1 Zhishu splits into computation and principle

The paper distinguishes:

```text
jisuan zhishu  = computational technique / practical calculation
shuli zhidao   = mathematical principle / abstract rule
```

This is useful because it separates executable procedure from symbolic doctrine.

#### 2.2 One / Two / Three are not merely counts

The paper gives layered number roles:

```text
One:
  taixu / great void → Chaos-One
  yuanqi             → Crack-One / original matrix
  wanwu              → generated beings

Two:
  liangyi / yinyang bisection
  infinite hierarchical subdivision
  relative classification by reference frame

Three:
  one divides into three
  heaven / earth / human
  sansheng / three-generative principles
```

The phrase **original matrix** for yuanqi is important. It is not a scalar origin. It is an origin-state from which classification and embodiment proceed.

#### 2.3 Yinyang classification is reference-conditioned

The paper explicitly says yinyang classification maintains determinacy under fixed reference conditions, but changes under altered time/spatial conditions or deeper subdivision.

Mechanism:

```text
fixed reference frame → determinate yinyang assignment
changed reference frame → localized reclassification
further time/space breakdown → new classification layer
```

This is a direct warning against static binary readings.

#### 2.4 Wuxing is nonlinear restoration, not static pentad symbolism

The paper describes wuxing generation/restriction as nonlinear and discontinuous. Abnormal states do not persist indefinitely; systemic chain reactions restore balance. From a limit perspective, equilibrium is a trend, not a permanent steady state.

Mechanism:

```text
relative balance
→ local imbalance
→ chain reaction
→ restored balance
→ never absolute final equilibrium
```

This is relevant to saturation/lift because it describes state correction through cyclic relational dynamics, not static category membership.

#### 2.5 The major new hit: Linggui Bafa as explicit selector grammar

The paper gives an executable medical selection rule:

```text
(day stem-branch + hour stem-branch) / 9 or 6
```

The divisor is not fixed. It is selected by the day class:

```text
yangri day → divide by 9
yinri day  → divide by 6
```

Example in the paper:

```text
ren = 6
wu  = 9
yi  = 9
chou = 10
sum = 34

yichou is yinri → divide by 6
34 mod 6 = 4
remainder 4 → substitute into jiugong
jiugong position → linqi acupuncture point
```

This is a real **state-conditioned selector**, not a vague numerological association.

#### 2.6 Central-five exception / deformation rule

The paper says Chinese medical systems reconcile mismatched numerical systems by artificial deformation, including:

```text
transform number five into six
or leave one position vacant
```

It then gives a special rule for remainder 5:

```text
remainder = 5 and male   → gongsun point
remainder = 5 and female → zhaohai point
```

This is important because the central state is not treated as a normal position. It triggers a branch or reallocation.

### Reunification

This paper gives the cleanest **external selector analogue** so far:

| Phase role | TCM / jiugong analogue |
|---|---|
| Q-admitted state | day class, time class, yinyang reference frame |
| B/legal refinement | sum and divide under selected modulus |


## 3. `Tao-Research/out/ChinaandUniversals.pdf`

Title: **China and Universals**  
Author: Bernard Paul Sypniewski  
Status after this pass: `COMPLETE_FIRST_PASS`

### Paper-level read

This paper is a useful negative-control document. It challenges the claim that the I Ching hexagrams are simply binary arithmetic in the Leibniz sense. This does not weaken the Phase prospect. It sharpens it.

### Core extractions

The paper argues:

```text
Binary arithmetic requires fixed place-value order.
Hexagrams are stacks of lines and do not intrinsically specify one unique place-value order.
Leibniz rotated hexagrams to impose a binary order.
The traditional Chinese reading direction can produce a different binary number.
Some rotations preserve relations, some do not.
```

It also stresses that the hexagram set is closed:

```text
six-line hexagrams cover 0..63 only if forced into binary arithmetic
expanding beyond that would require septagrams etc.
traditional Chinese use is limited to trigrams and hexagrams
```

The key result is:

```text
I Ching hexagrams are not best treated as open-ended binary arithmetic.
They are a closed six-line state field with transformations, interpretations, and ordering rules.
```

### Reunification

This is protective. It blocks the wrong route:

```text
WRONG:
  hexagrams = ordinary binary numerals

BETTER:
  hexagrams = finite closed six-line carrier states
              with orientation, line order, complement, mutual change, and re-chart rules
```

Phase comparison should therefore use the hexagram corpus as a **finite carrier grammar**, not as evidence that ancient China had ordinary binary computation.

| Phase role | Corrected Yijing analogue |

| B | changing-line or line-construction process only if rule-locked |
| L | rotation/re-chart changes projection; different orderings give different readouts |

### Status

```text

if dotted lines exist, flip them to their opposite line form
flipped result gives second/future hexagram
```

Moving/dotted line rule:

```text
three-of-a-kind coin cast → dotted line
solid dotted line flips to dashed
broken dotted line flips to solid
current hexagram → second hexagram
```

```text

→ line-state selection

→ first projection/readout
→ marked changing lines mutate
→ second projection/readout
```

The useful part is not divination content. The useful part is the two-stage readout:

```text

projection is read
marked internal lines alter state
new projection is read
```

| Phase role | I Ching manual analogue |
