## Method lock

Phase Calculus/QBL is not being treated as Taoist math, Chinese math, Buddhist math, solar physics, modular forms, or numerical-grid engineering. The external corpus is being normalized by function:

```text
retained carrier/state
-> admissible operation standard
-> refinement/transition
-> blockage, threshold, contradiction, or no-next-step event
-> re-chart/lift/new domain
-> projection/readout from retained state
-> invariant/custody preservation
```

A bridge is not weakened by missing literal labels. A bridge is weakened only by a clearly identified missing functional component, or by an explicit negative test showing that the mapping cannot preserve the required state, operation, threshold, lift, or projection discipline.

## ICPR mechanism extraction

### 1. Chinese logic: admissibility by standard, not arbitrary naming

The Chinese logic section gives a strong operational comparison layer through `fa`, the standard/model that determines whether a thing fits a name or classification.

```text
name/object pair
-> standard fa
-> kind lei
-> reason gu + general law li
-> admissible inference pattern
```

Phase comparison:

```text
Phase retained state
-> current admissibility condition
-> allowed operation
-> transition/lift/readout
```

The useful point is not that `fa = Q` or `fa = B`. The useful point is that the operation is not free. It is governed by a retained standard that decides whether the next application is valid.

### 2. `lei` kind as bridge-preservation criterion

The Chinese logic section treats sameness/difference by kind as the criterion that licenses inference. This is a strong analogue for bridge validation:

```text
Two cases can be cross-mapped only if the relevant kind-defining property is preserved.
```

For this project, that becomes:

```text
A Phase/external bridge is valid only if the functional kind is preserved:
  retained-state completeness
  admissibility law
  transition/refinement mechanism
  blockage/saturation criterion
  lift/re-chart behavior
  projection discipline
  invariant/custody preservation
```

This is a better rigor tool than literal name matching.

### 3. `xiao`, `pi`, `yuan`, `tui` as comparison operations

The Chinese logic chapter gives four usable reasoning operations:

```text
xiao: set up standard fa, test conformity.
pi: explain one thing using a different known thing.
yuan: adduce a case the opponent accepts and show same-kind structure.
tui: infer from accepted sameness into a not-yet-accepted target.
```

Operational reading for this research:

```text
xiao = define the bridge standard.
pi   = use an external paper as an explanatory analogue.
yuan = show the external case and Phase case are same functional kind.
tui  = transfer a tested structural claim across the bridge.
```

This directly supports the current methodology: do not reject a bridge because labels differ; test whether the kind-defining functional structure is preserved.

### 4. Finite memory / automata as QBL custody normal form

The automata section is a very strong formal normalization source:

```text
finite registers
-> finite configuration
-> event/action enabled at current configuration
-> transition/update
-> run word
-> accepting/final boundary
```

Phase comparison:

```text
Ξ̂ retained state
-> Q/B/L admissibility
-> selected operation
-> updated Ξ̂ + accrued word
-> saturation/lift/readout boundary
```

Key bridge: the process is not defined only by its visible output. It is defined by configurations and the word of events/transitions.

### 5. Enabled transition = admissible operation

The finite automata section explicitly uses the idea of an event/action being enabled at a state. This is very close to the QBL custody distinction:

```text
not every symbol is legal at every state.
```

For Phase comparison:

```text
B is enabled only under the current Q-defined admissibility/floor condition.
Q is enabled when B is blocked but the phase cycle is not complete.
L is enabled when continuation inside the current domain is saturated or would repeat/violate the admissible state.
```

This is an external formal source for the project’s key wording:

```text
retained state governs admitted operation.
```

### 6. Category/definability as normalization discipline

The model-theory/category-adjacent section gives another useful bridge:

```text
structure M
-> definable sets
-> definable maps
-> category Def(M)
-> quantifier elimination / normalization when available
```

Phase comparison:

```text
retained state system
-> observable/projectable structures
-> structure-preserving maps
-> quotient/readout functors
-> normalization before comparison
```

This supports the “after normalization of the problems, or through cross mapping” goal. It also gives an objective missing-piece criterion:

```text
A bridge remains incomplete until the normalization map is explicit enough to show what is retained, what is transformed, and what is forgotten.
```
