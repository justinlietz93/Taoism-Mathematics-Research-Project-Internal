### A1. Chinese logic: `fa`, `lei`, `ming`, `bian`, `xiao/pi/yuan/tui`

High-value extraction:

```text
Chinese logic is not presented as abstract theorem syntax only.
It is practical argumentation over names, kinds, standards, sameness/difference, and inference patterns.
```

Key operational structures:

```text
ming      = names / classification labels
lei       = kinds / sameness-of-kind and difference-of-kind
fa        = standard / model / rule of admissible classification
bian      = disputation / argumentation over opposed claims
xiao      = standard-conformity reasoning: set fa, test case against fa
pi        = illustrating by a known different thing with relevant similarity
yuan      = adducing / transferring accepted similar structure
tui       = inferring by using sameness between accepted and refused cases
```

```text

  retained state + admitted operation + projection/readout

  retained classification/kind structure + standard/fa + admissible inference pattern
```
