### 2. Four-tableau / 2+2 partition structure

The paper maps four cases into a two-bit basis:

```text
(u | S)      -> |00>
(u | S*)     -> |01>
(u* | S)     -> |10>
(u* | S*)    -> |11>
```

It then groups them into dual partitions:

```text
D  = |00> + |11>
D* = |01> + |10>
```

Functional value:

```text
This is a useful external model for retained channel partitioning.
It is not QBL by itself.
It supplies a clean way to keep observer-conditioned complement pairs separate from literal state complement pairs.
```

### 3. Operator commutation must be handled carefully

The paper explicitly treats cases where dual operations commute in the abstract parity algebra, but later distinguishes non-exchangeable not-operators in the Buddhist logic sections.

Functional value:

```text

```

For QBL research:

Functional value:

Functional value:
