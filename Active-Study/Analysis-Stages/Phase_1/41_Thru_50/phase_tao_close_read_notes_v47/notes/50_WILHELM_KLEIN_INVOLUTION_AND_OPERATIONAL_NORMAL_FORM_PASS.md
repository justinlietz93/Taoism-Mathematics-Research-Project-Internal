The Wilhelm King Wen adjacent-pair structure is not scalar-adjacent noise. It is generated by a small finite carrier-operation algebra on the retained six-line state.

For each odd/even adjacent pair, the odd six-line carrier `s` was tested against:

```text
I(s)  = s
R(s)  = reverse(s)
C(s)  = complement(s)
RC(s) = reverse(complement(s))
```

Result:

```text

anti_palindrome_R_equals_C: 4 pairs
palindrome_C_equals_RC: 4 pairs
other_or_unclassified: 0 pairs
```

All 32 adjacent pairs are explained by retained-carrier involutions.

This is a stronger result than the previous count of reverse/complement pair types because it identifies the exceptional cases as stabilizer collapses of the same finite operation family:

```text

  C(s) = RC(s)

anti-palindrome where R(s)=C(s):

```
