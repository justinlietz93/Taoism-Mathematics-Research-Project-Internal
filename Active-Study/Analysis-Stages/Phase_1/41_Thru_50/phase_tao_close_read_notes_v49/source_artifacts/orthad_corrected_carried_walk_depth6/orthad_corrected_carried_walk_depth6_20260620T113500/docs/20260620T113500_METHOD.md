# Method

1. Start with `q=(1,1)`, `theta_quarters=0`, `depth=0`, active axis `axis_0`.
2. Repeat for six domains.
3. In each domain, apply six exact primitive pairs:

```text
B: (u,v) -> sort(v, u+v)
Q: theta += π/2
```

4. Apply `L` after the six signed positions:

```text
latch active axis
append exactly one new active axis
carry theta
carry q
increment depth
```

5. Emit channel fields from the walk itself.
6. Only after emission, compare the internal sign to the external Shadow Residual character χ12.

## Explicitly forbidden and absent

```text
uv cutoff
candidate search
scalar score
capacity ladder after first doubling
q reset at L
theta reset at L
external χ12 inside emission
```
