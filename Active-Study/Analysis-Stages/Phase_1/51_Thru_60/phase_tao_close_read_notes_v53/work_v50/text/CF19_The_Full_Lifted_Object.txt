                                                      Abstract
        This paper develops the full lifted object for the Void Dynamics Model primitive roll and
        then uses it as a generator rather than as a stopping point. The guiding question is simple:
        when a visible phase returns to where it started, what state has actually returned? We
        answer by carrying the visible witness into the retained state Ξb n = (An , qn , θn , κn , cn ), where
        the host class, balanced arithmetic carrier, phase, completion index, and centered completion
        germ are all kept until projection is safe. This lifted object separates visible recurrence from
        exact state return, gives closed laws for quarter continuation, balanced refinement, and host
        transfer, and identifies the canonical balanced anchor (55, 89), with uv = 4895, germ width
        2π/4895, and completion coefficients 1/24 per edge and 1/12 two-sided. On the canonical
        launched family, the internally carried edge phases and completion coefficients generate the
        classical order-3 mock-theta branch pair as the untwisted calibration branch. The paper then
        pushes past that identification by deriving the host-twisted conjugate-edge branch family
        forced by the selector-closed macro-calculus acting on the retained germ under quotient
        descent; the primitive quarter-twisted member has coefficient prefix 1 + q − q 3 + q 4 + q 5 + · · ·
        and differs from the untwisted mock-theta branch already at coefficient q 2 . CF19 therefore
        closes exact return to lifted origin and opens a reusable extension mechanism: classical mock
        theta appears as one projection of the lifted branch grammar, while host-twisted branches
        become new VDM-native objects for downstream quotient descent and classification.

        Keywords: Void Dynamics Model, Phase Calculus, lifted state, exact return, balanced
        refinement, mock theta, completion germ, quotient descent, host-twisted branches

1   Introduction
A visible cycle is easy to recognize. A full return is more interesting: the visible point may come back while
the carried state has changed. This paper studies that distinction for the primitive VDM roll by keeping the
full lifted state until return can be decided coordinate by coordinate.

What this paper does
The paper has five jobs. First, it defines the full lifted object and the coordinatewise meaning of true
return. Second, it proves the closed balanced-window evolution law, including quarter continuation, balanced
refinement, and host transfer. Third, it identifies the first admissible balanced anchor (55, 89), records the
germ scale 1/4895, and gives the finite completion coefficients. Fourth, it shows how the launched lifted state
generates the mock-theta branch pair from internal VDM data before comparing it with classical notation.
Fifth, it uses the now-closed operator core, selector-closed macro-calculus, and quotient-descent machinery of
Phase Calculus to derive the beyond-classical host-twisted branch family forced by the retained two-edge
germ: the classical mock-theta branch is the untwisted member, and the primitive quarter-twisted branch is
the first new extension object.

From identification to extension
Earlier versions of CF19 naturally ended at the recognition of Ramanujan’s order-3 mock-theta branch. That
recognition is still important, but in the mature Phase Calculus setting it is no longer the endpoint. It is
a calibration point. The full lifted object, the operator core, the selector-closed macro-calculus, and the
quotient law
                                               Π◦E =G◦Π
make the branch generator portable. Once the two carried edge factors are visible, the selector-closed
macro-calculus fixes the transported two-edge shell at each realized depth, and quotient descent permits host
phase only as a conjugate action on those retained edge slots. The untwisted double-edge kernel (1 + q j )−2 is
therefore the trivial-host member of a forced host-twisted conjugate-edge family. The resulting VDM-native
q-series are produced by the same retained-state mechanism but need not collapse back to the classical
mock-theta function.

How to read the companion package
The main paper carries the definitions, theorems, and proof structure. The companion package supports
inspection: it contains the numerical traces, coefficient tables, source snippets, and a curiosity-driven notebook
that walks through the anchor, launch-family, completion-coefficient, and host-transport witnesses. Those
materials are useful for reproduction and extension, while the main argument remains readable on its own.
