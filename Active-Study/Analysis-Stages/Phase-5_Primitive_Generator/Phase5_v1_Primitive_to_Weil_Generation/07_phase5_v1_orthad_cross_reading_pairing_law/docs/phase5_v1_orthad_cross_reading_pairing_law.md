# Phase 5 v1 — Orthad Cross-Reading Pairing Law Attempt

## Claim attacked

The Orthad cross-reading law is generated by unary tension: a retained orientation is not read only against itself, but against its dual-bearing relation to other retained orientations through shared QBL history.

## Fixed canon constraints retained

1. The degenerate macro grammar `R=Q∘B, S=Q, T=L` remains suspended for Phase 5 generation testing.
2. Macro symbols do not collide with Weil generator symbols.
3. Scalar shadow readouts remain external, but retained orientation-residue vectors and couplings are not banned if they are derived.
4. No dense orientation block is admitted by hand.
5. Comparison is sealed: native tables are written and hashed before Weil targets are constructed.

## Primitive histories

For orientation index `r`, the native history used in this attempt is

```text
W_r = (BQ)^r
```

This is not a macro relabel. It is an explicit finite word in the primitive alphabet.

## Candidate native cross-read laws

The pass derived these candidates from the pair of histories `(W_r,W_s)`:

```text
commutator_defect        = 0
shared_prefix_count      = min(r,s)
shared_prefix_symbol_count = 2 min(r,s)
separation_defect        = |r-s|
latch_overlap_count      = min(floor(r/6), floor(s/6))

```

The strongest native discovery is `shared_prefix_count = min(r,s)`: it is a real off-diagonal cross-read because it compares two orientations through common QBL ancestry.

## Post-seal result

The frozen native pairings do not match the finite Fourier phase law, whose exponent is bilinear in `r` and `s`. The native pairings are overlap or separation laws. They are not bilinear on `Z/NZ`.

The frozen diagonal induced by these candidate pairings also does not produce the Gauss law `r^2`.

## Verdict

```text
BOUNDARY_NATIVE_CROSS_READ_FOUND_NOT_WEIL
```

This is not diagonal-only. It is not generated. It is a useful boundary: Orthad cross-reading exists natively as history-overlap, but the current QBL/Orthad rules do not yet contain the multiplicative cross-history operation needed for `r*s`.

## Exact missing object

A native rule that multiplies independent orientation histories rather than merely measuring common ancestry, separation, or commutator defect.
