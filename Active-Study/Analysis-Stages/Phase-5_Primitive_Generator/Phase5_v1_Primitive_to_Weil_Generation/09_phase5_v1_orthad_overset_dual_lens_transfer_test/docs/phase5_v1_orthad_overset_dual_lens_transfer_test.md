## Native derivation

Let the orientation histories be indexed by:

```text
W_r = (BQ)^r
```

The native successor is:

```text
sigma_N(W_r) = W_{r+1 mod N}
```

The plus chart basis is the point orientation basis:

```text
e_r
```

The complementary minus chart is the normalized eigen-cochain basis of `sigma_N`. A receiver cochain `chi_s` obeys:

```text
chi_s(sigma_N e_r) = lambda_s chi_s(e_r)
lambda_s^N = 1
```

Therefore:

```text
chi_s(e_r) = c_s lambda_s^r
```

Normalization gives:

```text
|c_s| = N^(-1/2)
```

The positive QBL successor orientation fixes:

```text
lambda_s = exp(-2*pi*i*s/N)
```

So the native overset transfer coefficient is frozen as:

```text
K_N(r,s) = N^(-1/2) exp(-2*pi*i*r*s/N)
```
