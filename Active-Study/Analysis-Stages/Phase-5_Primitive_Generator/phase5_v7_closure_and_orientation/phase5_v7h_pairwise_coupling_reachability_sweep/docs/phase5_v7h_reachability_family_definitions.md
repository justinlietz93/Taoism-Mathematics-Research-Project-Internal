# Reachability Family Definitions

Native shared-boundary histories produce products `(q_i+1)(q_j+1)` at retained latch events. v7h allows bounded sums/differences of such latch products, plus branch-commutator differences.

The unit-latch program is not an external dial: it is the repeated shared boundary with `q_i=q_j=0`, producing native increment `1` before comparison.
