# Multi-Axis Frontier Note

The doubled carrier correction fixes the missing-hand parity artifact for a single orientation axis.

It does not settle general finite quadratic modules.

A general carrier can have product structure:

```text
A = A_1 × A_2 × ... × A_k
```

and a quadratic form with cross-axis terms:

```text
q(x_1, x_2) = q_1(x_1) + q_2(x_2) + c x_1 x_2
```

The next test must decide whether Orthad overset transfer generates these cross-axis terms or only direct sums of independent cyclic axes.
