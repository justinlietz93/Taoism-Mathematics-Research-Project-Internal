# Phase 5 v7b: Doubled Orientation Carrier Test

Status: DOUBLED_ORIENTATION_CARRIER_RESOLVES_ODD_N_ARTIFACT

This package corrects the Phase 5 v7 odd-N boundary interpretation. The earlier test put the quadratic self-twist on a carrier of size N. The quadratic itself has denominator 2N, so the native carrier is the doubled orientation carrier D_N = Z/(2N)Z.

Result: when both orientation hands are retained, the quadratic self-twist is well-defined for every tested base modulus N. The previous odd-N failure is reclassified as a folded-carrier artifact, not a primitive boundary.
