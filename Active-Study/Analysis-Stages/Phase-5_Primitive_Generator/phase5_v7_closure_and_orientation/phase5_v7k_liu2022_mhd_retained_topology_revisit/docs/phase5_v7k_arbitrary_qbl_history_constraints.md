# Arbitrary QBL History Constraints from Liu 2022

This paper suggests the QBL normal-form problem must include:

1. retained branch identity,
2. boundary/latch event identity,
3. endpoint custody transition,
4. projection-specific visibility channel,
5. ambiguity between retained topologies with similar projections,
6. a data-completeness gate before stronger claims.

The paper cannot itself provide a QBL history normal form because its raw time-resolved field-line connectivity/animations are not available in this pass.
