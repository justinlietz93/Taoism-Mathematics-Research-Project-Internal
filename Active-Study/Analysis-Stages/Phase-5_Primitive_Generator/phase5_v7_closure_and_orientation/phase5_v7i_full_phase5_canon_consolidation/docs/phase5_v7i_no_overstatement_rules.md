# No-Overstatement Rules

Patch or reject stale wording that says:

```text
Odd N is a boundary.
Even carrier law.
Orthad is diagonal.
Single cyclic carrier proves product-module closure.
Hand-supplied c_ij proves native generation.
Triple latch incidence is an independent Weil tensor.
Scalar projection cancellation means no retained closure.
```

Use:

```text
Odd folded carrier failure is projection loss.
The retained cyclic carrier is Z/(2N)Z.
The Orthad is overset dual-lens.
Product modules require product-module tests and coupling tensor gates.
c_ij must be extracted and sealed before comparison.
Higher latch incidence must project to pairwise bilinear form or fail finite quadratic module gates.
Scalar projection cancellation is projection loss, not retained-carrier failure.
```
