# Phase 5 v1 overset dual-lens patch

## Canon addition

The Orthad lens has two lawful read modes:

1. same-chart self-read, preserving the existing diagonal lens;
2. overset dual-chart transfer, derived from the finite QBL orientation successor.

The off-diagonal transfer table is not stored inside one diagonal lens. It belongs to the chart-transfer map between complementary Orthad charts.

## Permanent rule

Do not add dense off-diagonal blocks to the same-chart lens. Derive transfer entries from the relation between charts.
